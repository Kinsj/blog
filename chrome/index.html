# 页面渲染机制（DOM和CSSOM树）

  1. 下载HTML，开始解析
  2. 解析HTML时检测到有外部资源依赖时启动别的线程下载
  3. 当外部资源是JS时，html解析会暂停，等JS下载完成后才会继续解析HTML
  4. HTML解析时会解析成DOM，并构建DOM树
  5. CSS下载完成后会开始解析CSS，CSS会解析成CSS对象，进而构建CSSOM树
  6. 两棵树都构建完成后，浏览器会根据这两棵树构建出一棵渲染树
  7. 渲染树构建完成后浏览器开始布局计算，计算元素大小和位置
  8. 布局计算完成后就会渲染页面，展示网页
